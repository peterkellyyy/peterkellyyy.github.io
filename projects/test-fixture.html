Load Test Fixture

A repeatable benchtop fixture for measuring component performance under controlled loading conditions.

Overview

Designed a fixture to locate components consistently, react load through a rigid frame, and leave access for instrumentation.

Engineering Focus

  • Established load paths and fixture stiffness targets.
  • Added locating features for repeatable setup between test runs.
  • Planned sensor placement and operator-safe access zones.

Outcome

The fixture concept gave the test team a practical path to collect comparable measurements across design iterations.

Project Details

Tools
CAD, load path review, drawing package
Role
Fixture design
Deliverables
Fixture assembly, setup notes, BOM
